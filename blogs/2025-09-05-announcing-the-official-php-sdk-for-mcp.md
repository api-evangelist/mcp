The official PHP SDK for the Model Context Protocol is now generally available. Built in collaboration with the PHP Foundation and Symfony , the PHP SDK handles protocol details, so developers don’t have to worry about low-level mechanics and can focus on building their applications. The initial release enables PHP developers to build MCP servers , exposing tools , prompts , and resources to AI applications. Support for PHP applications to act as MCP clients will follow.
